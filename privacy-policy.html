Сайт использует файлы cookie и аналогичные технологии для:

5. Локальное хранение данных

Сайт использует технологии локального хранения данных:

5.1. LocalStorage

Используется для хранения:

  • Истории поисковых запросов
  • Списка избранного контента
  • Пользовательских настроек интерфейса
  • Кэша данных для ускорения загрузки

5.2. IndexedDB

Используется для хранения:

  • Кэшированных результатов поиска
  • Информации об аниме
  • Больших объемов пользовательских данных

Все данные хранятся локально в Вашем браузере и не передаются на наши серверы.
